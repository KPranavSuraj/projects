The objective of the game is very simple: 
1) Player guesses letters to find a hidden word.
2)Each wrong guess reduces the number of attempts
3)The player wins the game if he guesses the word without losing attempts.
Game Works like this:
Ask the player for the input, in this case letter of the word.
If yes: reveal the letter, If no: reduce attempts and ask them to guess the next letter.
This process is continued until player guesses the correct word or player loses.
If word is fully guessed : Player Wins.
If word is not fully guessed : Player Loses.
